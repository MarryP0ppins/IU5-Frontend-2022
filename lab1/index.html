  • HTML

    HTML
    HTML — стандартизированный язык разметки документов для просмотра веб-страниц в браузере.

    HTML создаёт каркас сайта, а другие технологии используются для оформления внешнего вида (CSS) или контроля функционала страницы (JavaScript)
  • Теги HTML

    Теги HTML
    Теги - конструкции, которые указывают браузеру, как необходимо отображать содержащийся в них контент. У любого HTML-тега есть имя и он обернут в угловые скобки

    Можно выделить следующие категории:
    • Теги для текста (h1-h6, p, span и др.)
    • Теги для форм (form, input, label и др.)
    • Теги для таблиц (table, tr, td, th и др.)
    • Специальные теги (body, header)

  • Родительские и дочерние элементы

    Родительские и дочерние элементы
    В HTML в элементы можно вкладывать не только текст, но и другие элементы. В таком случае вложенный элемент называется дочерним, а тот, в который обёрнут дочерний - родительским.

    Дочерний элемент всегда должен завершаться раньше, чем заканчивается родительский.

  • Атрибуты

    Атрибуты
    Атрибуты задают дополнительные параметры элементов. Атрибут встраивается только в открывающий тег. Значение атрибута приводят в одинарных или двойных кавычках. Выбор кавычек значения не имеет, однако открывающие и закрывающие кавычки должны быть одинаковыми
  • Примеры использования тегов. Заголовки.

    Примеры использования тегов. Заголовки.
    В HTML существует шесть тегов заголовков разного уровня: h1, h2, h3, h4, h5, h6.

    Заголовок первого уровня — самый крупный, а последний - меньше обычного текста. Благодаря правильному использованию заголовочных тегов поисковым системам легче анализировать сайт, а пользователям будет удобнее его читать.

  • Блочные и строчные элементы

    Блочные и строчные элементы

    Элементы в HTML делятся на блочные и строчные:

    • Блочные элементы (block) занимают всю ширину своего родителя (контейнера), формально создавая «блок» (отсюда и название).
    • Строчные элементы (inline) это те, которые занимают только то пространство, которое ограничено тегами, определяющими строчный элемент (необходимое для отображения их содержимого) и не нарушая потока содержимого (не требующее новой строки после каждого элемента).

    Пример:

    Блочные: <div>, <p>, <ul>, <ul>, <section>

    Строчные: <span>, <a>, <strong>

  • Формы

    Формы
    Для создания форм используются следующие HTML-элементы: <form> <label> <input> <textarea> <button>.
